# Language Translator (Notebook Version)

This project is a Language Translator built for Jupyter Notebook using *Gradio* and *googletrans*.

## Features
- Translate text into multiple languages
- Text-to-Speech (Read Aloud) output

## Requirements
- gradio
- googletrans==3.1.0a0
- gTTS

## Run
Open the notebook and run all cells.
